Начальная установка:
1)Установить ImDiskTk с параметрами указаными на скриншоте img1.
2)Запустить из пуска ImDisk\Настройка Ram-диска.
3)Настроить как показано на скриншотах img2 и img3.
4)Нажать ОК, после открытия окна свойтв диска можно закрыть ImDisk.
5)Открыть пуск, Выполнить..., написать cmd, нажать ОК.
6)Нажать правой кнопкой мыши по заголовку консоли и выбрать умолчания.
7)На вкладке шрифт выбрать Lucida Console, нажать ОК, закрыть консоль.
8)Открыть config.lua текстовым редактором и поменять OutPath на нужный диск.

Запись программы на карту:
1)Записать программу на диск Z.
2)Запустить ImageWriter.
